# Raventracks: Experiments in Media

The site is hosted in Github Pages and visible at https://raventracks.org.

It serves as an online repository and access point for projects under development.

This is a statically maintained, handmade web site, although the pages on the site (web pages and PDFs) are created using power tools.

(Think of it as like ice sculpture or any other art made using machines.)

Everything here should be considered a 'proof' in the sense that no production infrastructure (no back end) is made available, thus the process is not reproducible.

For projects that aim to be accessible and reproducible *all the way down*, see links below.
