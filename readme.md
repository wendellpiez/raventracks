# Raventracks: Experiments in Media

The site is hosted in Github Pages and visible at https://raventracks.org.

It serves as an online repository and access point for projects.

This is currently a statically maintained, handmade web site, although many resources offered (web pages and PDFs) are created using power tools.

Think of it as like ice sculpture or any other art made using machines.

If you have a response or feedback to offer, please create a Github [Issue](./Issues/) or contact the proprietor directly at w e n d e l l p i e z (at) w a p i e z (dot) c o m .
